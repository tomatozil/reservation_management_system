예약 관리 시스템

서버가 한 대일 때 → 여러 대일 때 → DB가 여러 대일 때로 점진적으로 확장하며 동시성 문제를 인지하고 적절한 해결 방안을 적용했습니다.
테스트 코드와 테스트 툴(locust)을 사용하여 성능을 측정했습니다.

     synchronized로 어플리케이션 레벨에서 해결
     CAS 알고리즘을 활용한 스레드 세이프 자료구조로 어플리케이션 레벨에서 해결
